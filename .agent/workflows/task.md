# /task command

Use this command to break down a user story or feature into a structured epic with clear implementation steps.

## Steps

1. **Epic Definition**: Define the high-level goal and success criteria.
2. **Task Brakdown**: Break the epic into granular, manageable tasks.
3. **Implementation Plan**: Update the `implementation_plan.md` for the current ticket.
4. **TDD Sequence**: Plan the test-driven development sequence for each task.

Refer to `.agent/rules/task-creator.md` for task planning best practices.
