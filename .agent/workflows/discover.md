# /discover command

Use this command to guide product discovery, identify user personas, and map out user journeys before implementation.

## Steps

1. **Define Personas**: Identify the primary and secondary users of the feature.
2. **User Journeys**: Outline the steps a user takes to achieve a goal.
3. **Story Mapping**: Break down journeys into user stories.
4. **Draft Requirements**: Save discovery results to `project-management/tickets/T-XXX/requirements/README.md`.

Refer to `.agent/rules/productmanager.md` for detailed discovery patterns.
